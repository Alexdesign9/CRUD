Реализовать ведение каталога книг через веб-интерфейс.
 
Объект "Книга" характеризуется следующим набором параметров:
- жанр (детектив, мелодрама, фантастика и т.д.);
- автор;
- название;
- год выпуска.
 
Функции для работы с каталогом:
- просмотр списка;
- добавление записи;
- редактирование записи;
- удаление записи.

Все поля должны быть обязательными (при добавлении/редактировании), год не может быть больше текущего.
 
Технические требования:
1.Желательно использовать схему Model-View-Controller.
2.Серверная часть должна быть написана на PHP, c использованием ООП. Плюсом будет использование любых серверных фреймворков.
3.На стороне клиента нужно использовать JavaScript, использование любых фреймворков также будет плюсом.
4.Обновление страниц должно происходить без перезагрузки всей страницы (должна использоваться технология Ajax).
5.СУБД - MySQL, MSSQL.
6.Приложение желательно развернуть на внешнем сервере.
